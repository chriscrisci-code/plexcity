# Plexcity redesign

Static multipage redesign of [plexcity.org](https://plexcity.org/).

## Preview

Open `index.html` in a browser, or serve the folder with any static file server.

## Pages

- `index.html` — Home
- `about.html` — About
- `contact.html` — Contact
- `jobs.html` — Careers (links to ApplicantPro)
- `services/` — Accounting, IT, HR, Payroll
